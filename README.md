# 全球大气环流与东亚季风 —— 交互模拟器

面向**高中地理教学**的动态可视化工具，参考人教版高中地理选择性必修1。

> [!NOTE]
> 本项目是**教学简化模型**，旨在帮助理解大气环流机制，不是数值天气预报系统。

---

## 功能模块

| 模块 | 说明 |
|------|------|
| **太阳直射点周年运动** | 地球示意图上动态显示太阳直射点位置，支持时间轴拖动 |
| **气压带与风带模拟** | 全球纬度剖面图，气压带与风带随太阳直射点同步偏移 |
| **三圈环流可视化** | 哈德莱/费雷尔/极地环流的上升-高空输送-下沉-近地回流展示 |
| **东亚季风形成** | 亚洲大陆-太平洋简化地图，展示海陆热力差异驱动的夏/冬季风 |
| **雨带移动 (ITCZ)** | 降水带跟随太阳直射点的季节移动可视化 |
| **气候带联动** | 9 大气候带与气压带/风带联动显示，点击查看详细成因 |
| **考试模式** | 随机出题、自动判分、逐题反馈，适合课堂练习 |

## 使用说明

1. 启动后使用侧边栏滑块选择月份（支持连续小数，全年滑动）
2. 开启动画自动播放气压带/风带/雨带的季节变化
3. 点击「开始考试」进入随机出题模式
4. 在气候带下拉菜单中选择具体类型查看成因、分布、降水、气温

## 物理模型 (教学简化版)

- **太阳赤纬**: δ = 23.5° × sin(2π × (month − 3) / 12)
- **气压带偏移**: 随太阳直射点按不同比例偏移（ITCZ 100%、副热带 80%、副极地 50%、极地 10%），偏移逻辑基于气压带**类别标识**而非数值纬度
- **风带分类**: 基于气压带类别组合判定（equatorial_low + subtropical_high → 信风带；subtropical_high + subpolar_low → 盛行西风带；subpolar_low + polar_high → 极地东风带），不受偏移后纬度值影响
- **季风判定**: 区间判断（夏季风 5~9 月、冬季风 11~次年 2 月、其余过渡期），强度采用连续函数平滑动画
- **雨带位置**: 跟随太阳直射点，约 1 个月滞后

## 教学价值

- **三圈环流**: 直观展示 Hadley/Ferrel/Polar 环流的上升-高空输送-下沉-近地回流的完整结构
- **气压带风带季节移动**: 动态演示 ITCZ、副热带高压带、副极地低压带随季节的南北偏移
- **东亚季风成因**: 可视化海陆热力性质差异如何驱动夏季东南季风和冬季西北季风
- **雨带移动**: 展示 ITCZ 降水带与太阳直射点的联动关系
- **动态理解**: 替代静态课本插图，通过连续动画帮助建立空间-时间因果关系
